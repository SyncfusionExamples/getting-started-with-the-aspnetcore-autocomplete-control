# Getting Started with the ASP.NET Core AutoComplete Control

A quick-start project demonstrating how to integrate Syncfusion's ASP.NET Core AutoComplete control with support for local and remote data sources, custom styling, and flexible configuration.

## Features

- **AutoComplete Control**: Easy integration of Syncfusion's AutoComplete into ASP.NET Core projects
- **Data Source Support**: Both local list data and remote data source examples included
- **Customization**: Custom height and width configuration for suggestion lists
- **Bootstrap 5**: Built with Bootstrap 5 styling for modern UI
- **Production Ready**: Code snippets ready for production implementation

## Prerequisites

- Visual Studio 2022
- .NET Core SDK
- Syncfusion AutoComplete license or trial version
- Basic knowledge of ASP.NET Core and C#

## How to Run the Project

1. Checkout this project to a location on your disk
2. Open the solution file using Visual Studio 2022
3. Restore the NuGet packages by rebuilding the solution
4. Run the project

## Configuration

The AutoComplete control supports configuration through various properties including data binding options (local or remote), filtering and search behavior, custom styling and dimensions, and event handling callbacks. The component integrates seamlessly with ASP.NET Core dependency injection and provides extensive customization capabilities for different use cases and requirements.
